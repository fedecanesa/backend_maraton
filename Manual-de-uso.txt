Manual de uso

Carpetas
* src: Contiene todo el proyecto.
    * api: Contine las rutas del servidor
        * inicial.js : ruta inicial (/);
        * compras.js : rutas /compras
    * public: Contiene archvivos css, html
        * app.js: Donde escribo la aplicación

Archivos
* index.js : Arraca la aplicación
* compras.json : simula la base de datos donde contiene todas las compras.
